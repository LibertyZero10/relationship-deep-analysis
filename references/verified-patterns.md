## 使用方法

### v2.0 一键执行（推荐）

```bash

```

脚本内部已包含全部正则模式，无需手动复制。此文件作为：
1. 模式参考文档（了解每个框架匹配什么）
2. 指标计算公式表（阈值和评估标准）
3. 手动分析时的模式复制源

### 手动使用模式

```python
import re

# 编译模式
compiled = [re.compile(p) for p in patterns]

# 检查匹配
def match_any(text, compiled_patterns):
    return any(p.search(text) for p in compiled_patterns)

# 精确匹配（用于筑墙等短消息）
def match_exact(text, exact_list):
    return text.strip() in exact_list
```

## 关键指标计算公式

| 指标 | 公式 | 阈值 | 评估标准 |
|------|------|------|---------|
| 戈特曼比率 | 积极互动/消极互动 | >5:1 | >5=健康, 0.8-5=边缘, <0.8=危险 |
| 蔑视/批评比 | 蔑视频次/批评频次 | >1.0 | >1.0=危险区 |
| NVC成熟度 | 非暴力/(暴力+非暴力)×10 | >7 | >7=健康, 5-7=边缘, <5=不及格 |
| 交叉交易率 | 交叉/(交叉+互补)×100% | >30% | >30%=高冲突 |
| 依恋四象限 | 自我价值(正/负) × 他人信任(正/负) | - | 焦虑×回避=最差组合 |
| 信任总分 | (可预测+可依赖+信心)/3 | >7 | >7=健康, 4-7=边缘, <4=危险 |
| 情绪压抑率 | 压抑/(压抑+表达+重评+回避)×100% | <40% | >60%=危险(双方均高=最危险) |
| 消息变异系数(CV) | stdev/mean (日消息量) | <0.5 | >0.7=极不稳定 |
| 创伤绑定评分 | (间歇强化+极化+循环+权力)/4 | >6 | >6=中高度, >8=高度 |
| 惩罚性沉默率 | 惩罚性/(惩罚性+保护性)×100% | <50% | >80%=系统性情感虐待 |
| 关系韧性 | 七层均分 | >7 | >7=健康, 5-7=脆弱, <5=危险 |
| 分手-复合循环率 | 和好次数/分手次数×100% | - | >90%=分手已失去威胁性 |
| 追逐序列频率 | 追逐序列总数/天数 | - | 日均>10次=高焦虑 |
| BIS:BAS比值 | BIS频次/BAS频次 | ~1:1 | <0.5=BAS主导(冲动型), >2=BIS主导(回避型) |
| 消息归一化 | 指标频次/消息总数×1000 | - | 用于双方公平比较(消除消息量差异) |
